1.  Nombre de like d'un projet : "Select Count(user) from likeproject where project = ?;"
2.  Nombre de like d'un commentaire :  "Select Count(user) from likeproject where comment = ?;"
3.  Projets avec tels tag : "Select * from projet JOIN projet_tag ON projet_tag.id_projet = projet.id JOIN tag ON projet_tag.id_tag = tag_id where tag.title = ?;"
4.  Rôle d'un utilisateur : "Select role from user where id = ?;"
5.  Status d'un projet : "Select status from projet where id  = ?;" 
6.  Auteur d'un commentaire : "Select Pseudo from user JOIN comment ON user.id = comment.author;"
7.  Nombre de projet publié par un utilisateur : "select Count(*) From projet Where author = ?;"
8.  Obtenir les projets : "Select * from projet;"
9.  Obtenir un projet en particulier : "Select * from projet where id = ?;"
10. Compter le nombre d'utilisateur avec un pseudo et mdp (utile pour la connexion): "Select Count(*) from user where pseudo = ? and mdp = ?;"
11. Ajouter un projet : "Insert into projet(titre, content, status, coverUrl) Values(?, ?, Reviewing, ?);"
12. Supprimer un projet : "Delete from projet where id=?;"
13. Ajouter un commentaire : "Insert into comments(titre, content, status, coverUrl) Values(?, ?, Reviewing, ?);"
14. Supprimer un commentaire : "Delete from comments where id=?;"
15. Changer le status d'un projet : "Update projet SET status = ?;"
16. Moyenne des commentaires : "SELECT AVG(rating) FROM comment WHERE projet = ?;"
17. recherche mot : SELECT * From projet where titre Like "%'.?.'%"
